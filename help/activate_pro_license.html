Description

This tool can be used to activate a WbW-Pro license using the activation key issued at the time of purchase. Important information will be emailed to you, including your floating license ID, once activation is completed. Notice, only the tools within the WbW Pro (License required) toolbox require a WbW-Pro license. All other tools are within the freeware tier of WbW.
